# 🎓 Student Management System

A web-based Student Management System developed using Java EE technologies. This application allows students to manage their academic activities while enabling administrators to control course management and student registrations efficiently.

---

## 📌 Overview

This system provides a complete solution for managing students and courses within an educational environment. It includes role-based access for students and administrators, secure authentication, and dynamic course registration management.

---

## 🚀 Features

### 👨‍🎓 Student Functionalities
- Register and login securely  
- Update personal details  
- View available courses  
- Select and register for courses  
- View course registration status (Approved / Rejected)  

### 👨‍💼 Admin Functionalities
- Secure admin login  
- View all student records  
- Add new courses  
- Update or delete existing courses  
- Approve or reject student course registrations  
- Manage student enrollments  

### 🔐 Authentication & Security
- Role-based login system (Admin / Student)  
- Session management  
- Dashboard redirection based on user role  

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS  
- **Backend:** Java Servlets (Java EE)  
- **Database:** MySQL (phpMyAdmin)  
- **IDE:** Apache NetBeans  
- **Server:** GlassFish Server
